ngx-easymde

An Angular wrapper for the Easy MarkDown Editor originally derived from Angular for simplemde(Markdown Editor) by 卡色 cipchk.

The editor with default settings.

With a Customized Toolbar

Toolbar with only Bold, Italic, Heading toggle, Blockquote, Preview and Fullscreen buttons.

The editor can automatically save its contents to prevent changes when you reload the page.

With Event Handling
