ngx-easymde

Angular for Easy MarkDown Editor

Show Modal

Display the editor in a modal dialog.

With:


<button class="btn btn-primary" (click)="ngxSmartModalService.getModal('exampleModal').open()">Display Dialog</button>
<ngx-smart-modal #exampleModal identifier="exampleModal">
  <easymde [(ngModel)]="modal"></easymde>
  <button (click)="exampleModal.close()">Close</button>
</ngx-smart-modal>

Default Settings

The editor with default settings.

With:


<easymde [(ngModel)]="demo"></easymde>

Customized Toolbar

Limiting the toolbar to only Bold, Italic, Heading, Quote, and Preview.

With:


<easymde
  [(ngModel)]="customize"
  [options]="{{ '{' }} toolbar: ['bold', 'italic', 'heading', '|', 'quote', '|', 'preview'] {{ '}' }}">
</easymde>

Autosaving

The editor can automatically save its contents to prevent changes when you reload the page.

With:


<easymde id="ndx-easymde-autosave-demo" #easymde [(ngModel)]="autoSaving" [options]="autoSavingOptions"></easymde>

where autoSavingsOptions is defined as


autoSavingOptions = {{ '{' }}
  autosave: {{ '{' }} enabled: true, delay: 1000, uniqueId: 'ndx-easymde-autosave-demo' },
  renderingConfig: {{ '{' }}
    codeSyntaxHighlighting: true,
  },
};

Hidden toolbar and status bar

With:


<easymde [(ngModel)]="hiddenToolbar"
  [options]="{{ '{' }} toolbar: false, status: false, renderingConfig: {{ '{' }} codeSyntaxHighlighting: true } }"></easymde>

Event Handling

How to setup event handlers

With:


<easymde [(ngModel)]="eventcontent" (onChange)="onChange($event)" (onBlur)="onBlur($event)"></easymde>

and the handlers...


  onBlur(message: string):void {{ '{' }}
    alert(message);
  }

  onChange(message: string):void {{ '{' }}
    console.log(message);
  }

  <button class="btn btn-primary" type="submit" [disabled]="fg.invalid">Submit</button>
  <button class="btn" (click)="setDisabledForForm()">{{ '{{' }}this.buttonText}}</button>
